# Agent Skills | Notion

- แหล่งที่มา (ชั่วคราว): https://share.google/Up58eAM9d1Kd30FLx
- หมายเหตุ: ลิงก์ด้านบนยังไม่ใช่ลิงก์สาธารณะของ Notion/ไฟล์โดยตรง จึงยังดึงเนื้อหาไม่ได้ในระบบอัตโนมัติ

## วิธีทำให้ใช้งานได้ในระบบนี้
1) ถ้าเป็น Notion Page:
   - เปิด Share → Publish to web (หรือ Share → Anyone with the link → Can view)
   - ส่งลิงก์สาธารณะให้ผม หรืออัปเดตลิงก์ในไฟล์นี้แทนที่ลิงก์ชั่วคราว
   - ทางเลือก: Export เป็น Markdown/PDF แล้ววางไฟล์ไว้ในโฟลเดอร์นี้

2) ถ้าเป็น Google Docs/Sheets:
   - ตั้งค่า Share → Anyone with the link → Viewer
   - (Sheets) Publish to the web → เลือกเป็น CSV/เว็บเพจตามต้องการ
   - วางลิงก์สาธารณะไว้ในไฟล์นี้

## สถานะ
- ไฟล์นี้เป็นจุดอ้างอิงเก็บ “Agent Skills | Notion” ภายในโปรเจกต์
- เมื่อได้ลิงก์สาธารณะหรือไฟล์ Export แล้ว ผมจะอัปเดตให้แสดงผล/แนบไฟล์จริงทันที

## สารบัญ Skills (พร้อมใช้งานในระบบนี้)

- agent-browser — อัตโนมัติบนเว็บ: เปิดหน้า, ล็อกอิน, คลิก, กรอกฟอร์ม, ถ่ายภาพ, เก็บข้อมูล
- audit-website — สแกน SEO/Performance/Security 200+ ประเด็น พร้อมรายงานแก้ไข
- brainstorming — ระดมความคิด ออกแบบฟีเจอร์/เพจ กำหนดข้อกำหนดก่อนพัฒนา
- browser-use — ทดสอบเว็บเชิงสคริปต์: คลิก/กรอก/ดึงข้อมูลเป็นขั้นตอน
- copywriting — เขียน/ปรับปรุงข้อความการตลาด หน้าเว็บ อีเมล CTA
- find-skills — ค้นหาและติดตั้งสกิลเพิ่มเติมที่เหมาะกับงาน
- frontend-design — สร้าง/ปรับ UI ระดับ production (React/HTML/CSS)
- pdf — อ่าน/แยก/รวม/ใส่ลายน้ำ/ดึงรูป/ทำ OCR/สร้าง PDF
- remotion-best-practices — แนวทางสร้างวิดีโอด้วย Remotion (React)
- seo-audit — ตรวจ on-page/technical SEO รายหน้า พร้อมข้อเสนอแนะ
- skill-creator — สร้างสกิลใหม่/เวิร์กโฟลว์เฉพาะงาน
- supabase-postgres-best-practices — ออกแบบสคีมา/คิวรี/ปรับจูน Postgres
- ui-ux-pro-max — รีวิว UX/UI ให้เป็นมาตรฐาน ใช้งานง่าย และเข้าถึงได้
- vercel-composition-patterns — แพตเทิร์น component/composition ใน React
- vercel-react-best-practices — ปรับประสิทธิภาพ React/Next.js และ bundle
- vercel-react-native-skills — ทักษะสำหรับ React Native/Expo และประสิทธิภาพ
- web-design-guidelines — ตรวจโค้ด UI ให้สอดคล้องแนวปฏิบัติ/Accessibility
- xlsx — งานสเปรดชีต: เปิด/แก้/รวม/ทำความสะอาด/จัดรูปแบบ/ส่งออก

## ตัวอย่างการเรียกใช้งาน

- agent-browser: "เปิดหน้าเว็บไซต์ X → ล็อกอินด้วยบัญชีทดสอบ → ดาวน์โหลดรายงาน → ถ่ายสกรีนช็อตสรุป"
- xlsx: "อ่านไฟล์ A.xlsx และ B.xlsx แล้ว join ตาม order_id จากนั้นส่งออกเป็น results.xlsx"
- seo-audit: "ตรวจ SEO หน้า /pricing พร้อมสรุป Title, Meta, H1, ลิงก์เสีย และคำแนะนำเร่งด่วน"
